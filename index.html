The MIT-TTS Dataset

This is a creative-commons licensed speech dataset consisting of 1,540 short audio clips. Version 0.2 is of a single speaker lecturing in the MIT Opencourseware course "Biological Chemistry I." A transcription is provided for each clip. Clips vary in length from 1 to 13 seconds and have a total length of approximately 90 minutes.

The video lectures were published in 2013, and are creative-commons licensed. The audio was recorded by the MIT OpenCourseware project and is provided under a Creative Commons License.

File Format

The dataset is constructed to be identical to the LJSpeech dataset in order to be used as a drop-in replacement. This web page also uses LJ Speech as a template. Metadata is provided in metadata.csv. This file consists of one record per line, delimited by the pipe character (0x7c). The fields are:

  1. ID: this is the name of the corresponding .wav file
  2. Transcription: words spoken by the reader (UTF-8)
  3. Normalized Transcription: transcription with numbers, ordinals, and monetary units expanded into full words (UTF-8).

Each audio file is a single-channel 16-bit PCM WAV with a sample rate of 22050 Hz.
